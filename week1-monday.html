场景 01

AI对话场景：了解ChatGPT类工具的原理与使用方法

探索像DeepSeek、ChatGPT等大语言模型的工作原理，学习如何有效使用它们来提高工作效率和获取知识

工作原理

像DeepSeek、ChatGPT这样的大语言模型，本质上是通过预测下一个单词来生成文本的系统，它们的工作流程如下：

  1. 训练阶段：模型通过阅读互联网上的海量文本学习语言规律
  2. 理解输入：将用户提问转化为模型能理解的数字表示（向量）
  3. 生成响应：基于已学习的模式，推理和生成最可能的文本内容
  4. 上下文记忆：保留对话历史，使后续回答更加连贯和相关

关键概念

  • Token：模型处理的基本单位，可能是单词或词的一部分
  • Context Window：模型能记住的对话上下文长度
  • Prompt Engineering：优化提示以获得更好的模型回应

使用指南

1. 制定明确的提示词

  • 明确说明你的目标和任务
  • 提供必要的背景信息
  • 指定期望的输出格式

2. 迭代对话技巧

  • 从简单问题开始，逐步深入
  • 对模型的回答进行评估和修正
  • 使用思维链等提示词技术引导深入思考

3. 高级使用策略

  • 角色扮演：让AI扮演特定专家
  • 使用推理模型：学习AI思考过程
  • 多轮反馈：不断调整和改进输出

注意：模型有时会生成看似可信但实际不准确的内容（幻觉），重要信息需要进行验证。

提示词示例

你能解释一下量子计算的基本原理吗？请用通俗易懂的语言，假设我是一个高中生。

这个提示词明确指定了目标受众（高中生）和语言风格（通俗易懂），帮助AI调整解释的复杂度

角色扮演示例

我想让你扮演一位经验丰富的投资顾问。我正在考虑开始投资股票市场，但完全没有经验。请为我设计一个适合初学者的入门投资计划，包括风险管理策略和首批可考虑的投资类型。假设我有10万元资金，风险承受能力中等。

通过让AI扮演专家角色，并提供具体条件（资金、风险偏好），可以获得更有针对性的专业建议

DeepSeek界面与交互演示

高级提示词技巧

思维链（Chain of Thought）提示

我需要解决一个复杂的商业问题。我们的电子商务网站的购物车放弃率最近增加了15%。请一步步思考可能的原因，并为每个可能的原因提供具体的解决方案。分析时请考虑技术因素、用户体验和商业流程三个维度。

指导AI如何拆解分析问题，可以获得更深入、结构化的分析，而不仅仅是表面的答案

多轮迭代改进

第一轮：帮我起草一封面试后的感谢邮件模板，我申请的是市场营销经理职位。 第二轮：这封邮件看起来有点通用，能否增加一些更个性化的内容？请加入一些可以突出我对公司产品了解的部分，以及面试中讨论的三年营销战略计划。 第三轮：很好，但语气还是有点正式。能否调整为更加自信但不失专业的语气？同时，在结尾加入一个关于期待下一步的具体时间表的询问。

通过多轮迭代，您可以逐步完善AI的输出，从基础版本逐步调整到符合您特定需求的最终版本

场景 02

批判性思维场景：如何验证AI输出的可靠性

掌握对AI生成内容的批判性分析方法，学习如何判断信息的可靠性，并进行有效的事实核查

为什么批判性思维在AI时代尤为关键

在充斥着各种信息和观点的数字时代，批判性思维是我们区分事实与虚构、理性与煽动的核心能力。随着生成式AI的普及，这种能力变得更加重要。

批判性思维的价值

  • 在信息泛滥中筛选出真实可靠的内容，避免被错误信息或偏见所误导
  • 全面评估AI生成内容的可靠性和价值，成为AI的主人而非仆人
  • 减轻信息焦虑，知道大部分危言耸听的预测和宣传都经不起推敲
  • 提高解决问题的能力，对复杂问题形成更全面深入的理解

AI幻觉的原理与表现

AI幻觉是指AI模型生成看似可信但实际上不准确或完全虚构的内容。了解这一现象的原理，有助于我们更好地识别和应对它。

产生原因
  • 模型基于概率生成内容，而非检索确定的事实
  • 训练数据中可能包含错误或矛盾的信息
  • 模型倾向于生成流畅连贯的文本，即使需要"编造"细节
  • 复杂问题可能超出模型的知识范围或上下文理解能力
常见表现
  • 编造不存在的参考文献、数据或事件
  • 错误引用真实人物的言论或观点
  • 自信地提供看似精确但实际不正确的细节
  • 在不确定的问题上给出过度确定的回答

培养批判性思维的日常练习

批判性思维不是天生的，而是需要通过持续练习培养的能力。以下是一些可以融入日常生活的批判性思维训练方法：

质疑资讯源

每天选择1-2条社交媒体热门信息，追查其原始来源和证据链，评估可信度。

寻找反方论述

对你认同的观点，主动寻找理性的反对意见，尝试理解不同立场的合理性。

AI内容评估

要求AI分析可能存在偏见或错误的内容，然后评估AI的分析质量和盲点。

反思日记

每周记录一次你改变主意或发现自己错误的经历，分析思维过程和学习点。

AI辅助批判性思维实践

AI既是我们需要用批判性思维评估的对象，也可以成为培养批判性思维的助手。以下是几种结合AI进行批判性思维训练的方法：

新闻/社媒内容分析练习

每天选择一则新闻或社交媒体内容，与AI一起分析可能存在的偏见、不准确或误导性表述。这种练习帮助培养媒体素养，同时也锻炼你对AI回答的评估能力。

练习步骤
  1. 选择当天的一则新闻或社交媒体热门内容
  2. 请AI分析其中可能存在的偏见、误导或逻辑问题
  3. 评估AI的分析是否合理，是否遗漏了重要角度
  4. 思考如何验证文章中的关键事实声明
  5. 记录你的思考过程和从这次分析中学到的洞见
分析要点
  • 信息源的可靠性和可能的利益关系
  • 论据与结论之间的逻辑关系
  • 情感化语言和框架选择的影响
  • 重要背景信息的省略或不当强调
  • 统计数据的正确使用与解释

AI交叉验证练习

向多个AI工具提出同一问题，对比它们的回答，寻找共识点和分歧点。这种方法有助于识别AI幻觉，培养对AI回答的健康怀疑态度。

有效问题类型
  • 关于争议性话题的多角度分析
  • 特定学术概念或理论的解释
  • 历史事件的影响评估
  • 技术或产品的优缺点比较

批判性思维训练提示词

新闻/社媒内容分析

请帮我分析以下新闻/社交媒体内容可能存在的偏见、误导或逻辑问题。我希望你能从多个角度进行客观分析，包括： 1. 信息源的可靠性和潜在利益关系 2. 表述中的情感化语言或框架选择 3. 论据与结论之间的逻辑关系 4. 重要背景信息的省略 5. 统计数据的使用是否恰当 6. 可能存在的确认偏误或其他认知偏见 内容如下： "[粘贴新闻或社交媒体内容]" 在你的分析后，请提供2-3个问题，这些问题可以帮助我进一步验证或探索这则信息的真实性和全面性。

AI幻觉测试与评估

我想测试你对[特定主题]的知识准确性，并练习我的批判性思维能力。请回答以下问题： [关于该主题的具体问题] 在你回答的同时，请： 1. 标明哪些信息你非常确定（90%以上的把握） 2. 哪些信息你中等确定（60-90%的把握） 3. 哪些信息你不太确定（低于60%的把握） 4. 明确指出任何你不知道或无法确定的部分 此外，请提供2-3个方法，说明我如何可以验证你提供的关键信息的准确性。

场景小结

批判性思维是AI时代的必备能力，它不仅帮助我们有效利用AI工具，也是防止被错误信息和AI幻觉误导的关键防线。通过持续的练习和反思，我们可以：

  • 更有效地评估信息的可靠性和价值
  • 识别媒体报道和社交内容中的偏见和误导
  • 明智地使用AI工具，既能发挥其优势，又能规避其局限
  • 减轻信息焦虑，建立对复杂世界的理性认知
  • 发展终身学习的能力，不断调整和更新自己的认知框架

最重要的是，批判性思维不是孤立的智力活动，而是一种生活态度——保持好奇、开放和谦逊，愿意质疑自己的假设，并在证据面前调整观点。在AI加速发展的时代，这种思维习惯将成为我们最可靠的导航工具。

"在信息的迷雾中，批判性思维是照亮前路的灯塔；在AI的浪潮里，它是确保我们不被淹没的救生衣。"

周一任务

批判性思维与AI助手使用实战

通过实际操作多种AI工具，培养批判性思维和工具对比能力，提升AI应用效率

任务目标

  • 熟悉DeepSeek、豆包等AI工具的基本界面与功能
  • 通过实际使用，发现AI之间的差异，形成多工具对比思维
  • 训练AI任务拆解能力，观察AI在不同模式下的输出质量
  • 学习识别AI生成内容中可能存在的错误、偏见和"幻觉"

思维训练任务（10min）

1

AI内容交叉验证练习

向2-3种不同的AI工具（如ChatGPT、DeepSeek等）提出相同的问题，对比它们的回答，找出相同点和不同点。

建议选择专业领域或争议性话题作为问题

2

新闻/社媒内容分析

选择一则当天的热门新闻或社交媒体内容，使用批判性思维模板对其进行分析，识别可能的偏见或误导。

使用场景2中提供的分析提示词作为工具

日常任务内容（20min）

1

熟悉DeepSeek界面功能

  • • 逐个点击每个按钮，查看功能（例如：联网搜索、深度思考模式等）
  • • 记录自己觉得最有用/最难理解的功能点
2

注册另一个AI工具（豆包）

  • • 对比DeepSeek与豆包的界面，找出相似点和差异点
3

同一个问题，用不同方式问DeepSeek

  • • 问题示例："AI是什么？"
  • • 方式1：使用"深度思考"模式
  • • 方式2：使用"联网搜索"模式
  • • 方式3：都不用，直接提问
  • • 观察AI的回答有什么区别
4

同时在DeepSeek和豆包上问同样的问题

  • • 记录它们的回答，比较两者的优缺点
5

测试AI生成的文字在不同文档工具中的兼容性

  • • 复制AI回答，粘贴到飞书文档和Word文档
  • • 检查是否格式保持一致，有无丢失/错乱

交付物

  • 一份AI工具对比报告（可以用Word、Notion、飞书文档整理）
    • • 记录自己对各个工具的初步感受
    • • 回答这些问题：哪些功能是你最喜欢的？哪些功能有待优化？哪个AI在回答问题上更精准？AI生成的文本格式是否保持一致？
  • 一份新闻/社媒内容的批判性分析（400-600字），包括信息源评估、偏见识别和背景补充

注意事项

  • • 所有问题都要开新的对话，避免AI受到历史聊天的影响
  • • 以开放、客观的态度进行分析，避免自身确认偏误的影响
  • • 记录分析过程中的思考和观察，而不仅仅是结论
  • • 不要过分依赖AI的分析结果，练习自己独立思考的能力
